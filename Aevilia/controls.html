Aevilia's controls

Aevilia is a keyboard-controlled game. Two more features are planned : custom key binding and game controller support.
Aevilia, as of version 0.4.0, has light mouse support, but more is being added.

Key bindings

KeyAction
ZInteract, select option, advance text.
WCancel, display all text at once.
EOpen, close inventory
Left Shift(Keep pressed) Sprint : double walking speed.

Debug keys

KeyAction
P(Keep pressed) Hide debug information (if enabled with "-d" command-line option)
OFreeze game.
QResume game frozen by O. If O is still pressed, advances only one frame.
